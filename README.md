# ❌⭕ Tic Tac Toe – Interactive Web Game

A modern and responsive Tic Tac Toe web application built using HTML, CSS, and JavaScript. The project implements complete game logic, turn-based gameplay, winner detection algorithms, draw handling, game-state management, and an engaging user interface.

## 🚀 Live Demo

**Website:** https://tiktaktoegame99.netlify.app/

---

# 📖 Project Overview

This project recreates the classic Tic Tac Toe game in a modern web environment, allowing two players to compete in a turn-based experience.

The application manages player turns, validates moves, detects winning combinations, highlights winning cells, handles draw scenarios, and provides options to reset or start a new game.

The primary goal of this project was to strengthen JavaScript problem-solving skills, state management concepts, DOM manipulation techniques, and responsive user interface development.

# 🔥 Core Functionalities

## Turn-Based Gameplay

Players alternate turns between:

* Player X
* Player O

The turn indicator updates automatically after every move.

---

## Winner Detection Algorithm

The application evaluates all possible winning combinations:

```text id="gk912m"
Rows:
[0,1,2]
[3,4,5]
[6,7,8]

Columns:
[0,3,6]
[1,4,7]
[2,5,8]

Diagonals:
[0,4,8]
[2,4,6]
```

When a winning combination is detected:

* The winner is announced
* Winning cells are highlighted
* Further moves are disabled

---

## Draw Detection

The game automatically identifies situations where:

* All 9 cells are filled
* No winning combination exists

A draw message is displayed and gameplay ends.

---

## Game Reset System

Users can:

* Reset the current board
* Start a completely new game
* Instantly replay after a win or draw
